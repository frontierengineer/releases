# Frontier 0.1.1

Release-pipeline and distribution housekeeping on top of 0.1.0.

- Desktop app + host now ship in lockstep across every channel (nightly, rc,
  stable), each promoting the same validated build.
- Release tags reorganized: the desktop app owns the channel tags
  (`stable` / `rc` / `nightly`); the host bundle moves to `host-stable` /
  `host-rc` / `host-nightly`. Download links and the native installer updated to
  match.
- Signed + notarized macOS builds (Apple Silicon + Intel), plus Linux
  (AppImage/deb) and Windows installers.

No behavioural changes to the product itself versus 0.1.0.
